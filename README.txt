Girls Who Code Summer Immersion Program Application 2018

DESCRIPTION: Rock Paper Scissors utilizes sockets to allow players on distinct computers to play digital rock paper scissors. This is a command line based game.

HOW TO PLAY: Install Python3 and Pygame (pip install Python3 and Pygame). To run the file using terminal, navigate into the game directory, and simply run "python3 main.py". 

NOTE: To run the server and the clients on separate computers, add your server’s IP address as HOST in “userPort.py” (line 4).
